# Akshara Backend

Akshara is an ongoing project built for **Sahitya**, the literature club of **MVJ College of Engineering**.

Akshara Backend is a RESTful API built using **Node.js, Express.js, and MongoDB**.  
It serves as the backend for the Akshara web application, handling authentication, data management, and communication with the frontend.

---

## Tech Stack

- **Runtime:** Node.js  
- **Framework:** Express.js  
- **Database:** MongoDB (Mongoose ODM)  
- **Authentication:** JWT-based authentication  
- **API Style:** REST  
- **Version Control:** Git  

---

## Features

- User authentication and authorization  
- Secure REST APIs  
- CRUD operations on core resources  
- MongoDB schema design using Mongoose  
- Centralized error handling  
- Environment-based configuration  

---

## Phase 1 – Core Backend Implementation

- Designed REST API structure and routing
- Implemented authentication and authorization flow
- Created MongoDB schemas and models
- Integrated backend APIs with frontend requirements
- Handled input validation, error responses, and edge cases
- Structured the project for scalability and maintainability

### Implemented Functionalities
- User registration and login
- Uploading and managing **Books**, **Videos**, and **Posts** (tweet-like messages)
- Creating and managing video playlists
- Dashboard access for channel statistics
- Like functionality for books, videos, and posts
- Commenting on videos and posts
- View count tracking for books, videos, and posts
- User subscription / follow system
